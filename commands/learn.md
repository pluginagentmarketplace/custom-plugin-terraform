Start your personalized learning journey through 65+ developer roadmaps.

## Choose Your Path

### Beginner (0-6 months)
- **Programming Fundamentals** - Learn your first language
- **Databases** - Understand SQL basics
- **Web Fundamentals** - HTML, CSS, JavaScript basics

### Intermediate (6-12 months)
- **API Development** - Build REST APIs
- **System Architecture** - Design applications
- **DevOps Basics** - Docker and deployment

### Advanced (12+ months)
- **Caching & Performance** - Optimize systems
- **Distributed Systems** - Large-scale design
- **Security** - Build secure applications

## Getting Started

1. Choose your specialization (7 available)
2. Pick your starting level
3. Follow the curated roadmap
4. Build 5-10 projects
5. Progress to next level

Use `/browse-agent` to see all specialist agents!
Use `/projects` to see hands-on projects!
Use `/assess` to evaluate your knowledge!
