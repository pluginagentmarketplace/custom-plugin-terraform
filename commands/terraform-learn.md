# /terraform-learn

Master Terraform from basics to production-grade infrastructure.

## Learning Paths

### Beginner Track (4-6 weeks)
1. **HCL Fundamentals** (1 week)
   - Syntax and structure
   - Resources and variables
   - Basic expressions

2. **Terraform Workflow** (1 week)
   - init, plan, apply, destroy
   - State files
   - Local deployments

3. **Simple Infrastructure** (2-3 weeks)
   - VPCs and networking
   - EC2 instances
   - Security groups
   - Basic modules

4. **First AWS Deployment** (1 week)
   - Small 3-tier application
   - Remote state setup
   - Simple CI/CD

### Intermediate Track (8-12 weeks)
1. **Advanced HCL** (2 weeks)
   - Dynamic blocks
   - Complex expressions
   - For loops and conditionals

2. **Module Design** (3 weeks)
   - Module structure
   - Reusable modules
   - Module testing

3. **State Management** (2 weeks)
   - Remote state
   - Locking
   - Backup strategies

4. **Terraform Cloud** (2 weeks)
   - VCS integration
   - Team workflows
   - Cost estimation

5. **Multi-Environment Setup** (2 weeks)
   - Workspaces
   - Environment separation
   - Shared modules

### Advanced Track (12-16 weeks)
1. **Provider Development** (4 weeks)
   - SDK fundamentals
   - Resource implementation
   - Testing frameworks

2. **Enterprise Patterns** (3 weeks)
   - Multi-cloud
   - Governance
   - Scaling strategies

3. **Custom Modules** (2 weeks)
   - Complex composition
   - Registry publishing
   - Versioning

4. **Advanced Operations** (3 weeks)
   - Disaster recovery
   - Cost optimization
   - Compliance
