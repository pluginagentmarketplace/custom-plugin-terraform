## 1. üî§ Programming Fundamentals

**Duration**: 12-24 weeks (depending on level)
**Primary Target**: All developers
**Requirement**: None (foundational)

### Beginner Path (12 weeks)

**Weeks 1-2: Language Setup & Basics**
- Choose primary language (Python recommended)
- Syntax and basic operations
- Data types (int, float, string, bool)
- Variables and assignment
- Projects: Hello World, basic calculations

**Weeks 3-4: Control Flow**
- If/else statements
- Loops (for, while)
- Break and continue
- Logic and boolean operations
- Projects: Grade calculator, number guessing game

**Weeks 5-6: Functions & Scope**
- Function definition and calling
- Parameters and return values
- Variable scope (local, global)
- Recursion basics
- Projects: Function library, factorial calculator

**Weeks 7-8: Data Structures**
- Lists and arrays
- Dictionaries and maps
- Sets and tuples
- Operations and methods
- Projects: List operations, dictionary lookups

**Weeks 9-10: Object-Oriented Programming**
- Classes and objects
- Attributes and methods
- Constructors and destructors
- Inheritance basics
- Projects: Class design, simple game objects

**Weeks 11-12: Advanced Concepts & Testing**
- Exception handling
- File I/O
- Unit testing basics
- First complete project
- Projects: Complete application, test suite

### Intermediate Path (16 weeks)

**Weeks 1-3: Second Language**
- JavaScript or Go (contrast to first language)
- Idioms and language-specific features
- Async/await patterns
- Functional vs imperative

**Weeks 4-8: Algorithms & Data Structures**
- Sorting algorithms (merge, quick, heap sort)
- Searching algorithms
- Trees and graphs
- Hash tables and heaps
- Time and space complexity

**Weeks 9-12: Design Patterns**
- Creational (Singleton, Factory, Builder)
- Structural (Adapter, Decorator, Facade)
- Behavioral (Observer, Strategy, State)
- Pattern selection and use cases

**Weeks 13-16: Clean Code & Best Practices**
- SOLID principles
- Code organization
- Documentation and comments
- Refactoring techniques
- Projects: Refactor beginner project, complex algorithm implementation

### Advanced Path (24 weeks)

**Weeks 1-4: Third Language**
- Rust or Java (ecosystem understanding)
- Advanced language features
- Performance characteristics
- Ecosystem and libraries

**Weeks 5-12: Competitive Programming**
- LeetCode Medium problems (30+)
- LeetCode Hard problems (20+)
- Problem-solving strategies
- Time optimization techniques

**Weeks 13-20: Performance & Optimization**
- Profiling and benchmarking
- Memory optimization
- Algorithmic improvements
- Real-world optimization

**Weeks 21-24: Open Source & Mentoring**
- Contribute to major projects
- Code review and feedback
- Teaching others
- Advanced architectural patterns
