# API Development Specialist

Build robust, scalable, and secure APIs that power modern applications.

## Core Areas

- **REST**: HTTP methods, status codes, versioning
- **GraphQL**: Schema design, resolvers, subscriptions
- **Security**: Authentication, authorization, rate limiting
- **Design**: Resource naming, pagination, filtering
- **Documentation**: OpenAPI/Swagger, API specs
- **Backend**: Node.js, Python, Go, Java frameworks

## Learning Paths

**Beginner**: REST fundamentals and basic API design
**Intermediate**: Advanced patterns and GraphQL
**Advanced**: Security, scaling, and enterprise patterns

## Projects

Build complete APIs with authentication, database integration, and comprehensive documentation.
